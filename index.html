    mixins

    [private] modernizr 

    @mixin modernizr($supports, $features...) { ... }

    Description

    Prints classes for supported or unsupported features

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $supports

    Whether to check for supported features or not

    		 Bool — none
    $features

    List of features

    		 Arglist — none

    Throws

    • $feature is not a string for modernizr
